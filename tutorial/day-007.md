下面给你一个**可以直接照做**的 Day 7「整合日 Mini Bring-up」方案：用 **turtlesim + 你自己写的 bringup 包**把 topic/service/action/launch/rosbag/tf2 串起来，最后产出 README + demo.mp4 + retrospective。

> 你会得到一个最小系统：
>
> * **Topic**：/turtle1/cmd_vel（控制） + /turtle1/pose（状态）
> * **Service**：/reset（重置 turtlesim，现成的） + /week0/get_pose（你写的“查询状态”）
> * **Action**：/week0/drive_square（你写的“跑一个长任务：走方形轨迹”）
> * **Launch**：一键启动全部节点
> * **Rosbag**：录制并回放 /cmd_vel /pose /tf
> * **TF2**：发布 map → turtle1 的 transform，并用 tf2_echo / view_frames 展示 frame 关系

### 2.1 节点1：Pose Logger（订阅状态 /turtle1/pose）

`week0_bringup/pose_logger.py`

```bash
cat > week0_bringup/pose_logger.py << 'EOF'
import rclpy
from rclpy.node import Node
from turtlesim.msg import Pose

class PoseLogger(Node):
    def __init__(self):
        super().__init__('week0_pose_logger')
        self.declare_parameter('throttle_sec', 0.5)
        self._throttle = float(self.get_parameter('throttle_sec').value)

        self._last_log_time = self.get_clock().now()
        self.create_subscription(Pose, '/turtle1/pose', self.cb, 10)

        self.get_logger().info('PoseLogger started. Subscribing to /turtle1/pose')

    def cb(self, msg: Pose):
        now = self.get_clock().now()
        dt = (now - self._last_log_time).nanoseconds / 1e9
        if dt >= self._throttle:
            self._last_log_time = now
            self.get_logger().info(f'pose: x={msg.x:.2f}, y={msg.y:.2f}, theta={msg.theta:.2f}')

def main():
    rclpy.init()
    node = PoseLogger()
    try:
        rclpy.spin(node)
    finally:
        node.destroy_node()
        rclpy.shutdown()

if __name__ == '__main__':
    main()
EOF
```

**它满足：Topic 订阅状态**。

### 2.3 节点3：TF Broadcaster（发布 map → turtle1）

`week0_bringup/tf_broadcaster.py`

```bash
cat > week0_bringup/tf_broadcaster.py << 'EOF'
import math
import rclpy
from rclpy.node import Node
from turtlesim.msg import Pose
from geometry_msgs.msg import TransformStamped
from tf2_ros import TransformBroadcaster

class TurtleTFBroadcaster(Node):
    def __init__(self):
        super().__init__('week0_tf_broadcaster')
        self.declare_parameter('parent_frame', 'map')
        self.declare_parameter('child_frame', 'turtle1')

        self.parent = self.get_parameter('parent_frame').value
        self.child = self.get_parameter('child_frame').value

        self._br = TransformBroadcaster(self)
        self.create_subscription(Pose, '/turtle1/pose', self.pose_cb, 10)

        self.get_logger().info(f'TF broadcaster started. Broadcasting {self.parent} -> {self.child}')

    def pose_cb(self, msg: Pose):
        t = TransformStamped()
        t.header.stamp = self.get_clock().now().to_msg()
        t.header.frame_id = self.parent
        t.child_frame_id = self.child

        # turtlesim: x,y in 2D plane
        t.transform.translation.x = float(msg.x)
        t.transform.translation.y = float(msg.y)
        t.transform.translation.z = 0.0

        # yaw (theta) -> quaternion (z,w)
        yaw = float(msg.theta)
        t.transform.rotation.x = 0.0
        t.transform.rotation.y = 0.0
        t.transform.rotation.z = math.sin(yaw / 2.0)
        t.transform.rotation.w = math.cos(yaw / 2.0)

        self._br.sendTransform(t)

def main():
    rclpy.init()
    node = TurtleTFBroadcaster()
    try:
        rclpy.spin(node)
    finally:
        node.destroy_node()
        rclpy.shutdown()

if __name__ == '__main__':
    main()
EOF
```

**它满足：tf2 至少两个 frame 的关系**（`map` 和 `turtle1`）。

### 2.4 节点4：Action Server（长任务：驱动 turtle 走方形）

`week0_bringup/drive_square_action_server.py`

```bash
cat > week0_bringup/drive_square_action_server.py << 'EOF'
import math
import time

import rclpy
from rclpy.node import Node
from rclpy.action import ActionServer, CancelResponse
from rclpy.callback_groups import ReentrantCallbackGroup
from rclpy.executors import MultiThreadedExecutor

from geometry_msgs.msg import Twist
from turtlesim.msg import Pose
from week0_interfaces.action import DriveSquare


class DriveSquareActionServer(Node):
    def __init__(self):
        super().__init__('week0_drive_square_server')
        self._cb_group = ReentrantCallbackGroup()

        self._last_pose = Pose()
        self.create_subscription(Pose, '/turtle1/pose', self.pose_cb, 10, callback_group=self._cb_group)
        self._cmd_pub = self.create_publisher(Twist, '/turtle1/cmd_vel', 10)

        self._action_server = ActionServer(
            self,
            DriveSquare,
            '/week0/drive_square',
            execute_callback=self.execute_cb,
            cancel_callback=self.cancel_cb,
            callback_group=self._cb_group
        )

        self.get_logger().info('DriveSquare ActionServer started. Action: /week0/drive_square')

    def pose_cb(self, msg: Pose):
        self._last_pose = msg

    def cancel_cb(self, goal_handle):
        self.get_logger().warn('Cancel request received.')
        return CancelResponse.ACCEPT

    def stop(self):
        self._cmd_pub.publish(Twist())

    def execute_cb(self, goal_handle):
        goal = goal_handle.request
        side = max(0.1, float(goal.side_length))
        v = max(0.1, float(goal.linear_speed))
        w = max(0.1, float(goal.angular_speed))

        forward_time = side / v
        turn_time = (math.pi / 2.0) / w

        self.get_logger().info(
            f'Executing DriveSquare: side={side:.2f}, v={v:.2f}, w={w:.2f} | '
            f'forward_time={forward_time:.2f}s, turn_time={turn_time:.2f}s'
        )

        start = self.get_clock().now()
        dt = 0.05  # 20Hz
        steps = [
            ('forward', forward_time),
            ('turn', turn_time),
            ('forward', forward_time),
            ('turn', turn_time),
            ('forward', forward_time),
            ('turn', turn_time),
            ('forward', forward_time),
            ('turn', turn_time),
        ]

        for i, (mode, duration) in enumerate(steps):
            t0 = time.time()
            while (time.time() - t0) < duration:
                if goal_handle.is_cancel_requested:
                    self.stop()
                    goal_handle.canceled()
                    result = DriveSquare.Result()
                    result.success = False
                    result.message = 'Canceled'
                    result.final_x = float(self._last_pose.x)
                    result.final_y = float(self._last_pose.y)
                    result.final_theta = float(self._last_pose.theta)
                    return result

                cmd = Twist()
                if mode == 'forward':
                    cmd.linear.x = v
                    cmd.angular.z = 0.0
                else:
                    cmd.linear.x = 0.0
                    cmd.angular.z = w

                self._cmd_pub.publish(cmd)

                fb = DriveSquare.Feedback()
                fb.step = i
                fb.elapsed = (self.get_clock().now() - start).nanoseconds / 1e9
                fb.x = float(self._last_pose.x)
                fb.y = float(self._last_pose.y)
                fb.theta = float(self._last_pose.theta)
                goal_handle.publish_feedback(fb)

                time.sleep(dt)

        self.stop()
        goal_handle.succeed()

        result = DriveSquare.Result()
        result.success = True
        result.message = 'Completed square'
        result.final_x = float(self._last_pose.x)
        result.final_y = float(self._last_pose.y)
        result.final_theta = float(self._last_pose.theta)
        return result


def main():
    rclpy.init()
    node = DriveSquareActionServer()
    executor = MultiThreadedExecutor(num_threads=2)
    executor.add_node(node)

    try:
        executor.spin()
    finally:
        executor.shutdown()
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
EOF
```

**它满足：Action 长任务（走方形轨迹）**，同时它也会向 `/turtle1/cmd_vel` 发控制命令（Topic 发布控制）。

### 7.2 回放（让 turtlesim “吃回放数据”再次动起来）

确保 bringup 还在运行（尤其 turtlesim_node 还在）。

回放：

```bash
ros2 bag play bags/week0_demo
```

你会看到 turtlesim 里乌龟按录制的 cmd_vel 再走一次；PoseLogger 也会继续打印。

> **sqlite3 vs mcap（顺便复盘你 Day5 的问题）**
>
> * ROS2 默认常见是 **sqlite3**（`.db3`），优点是生态成熟、随机访问/索引方便。
> * **MCAP** 更偏“高吞吐日志容器”，跨工具链（ROS/非ROS）更友好，文件结构对流式写入/读取更现代。
> * 你可以试试（前提是你系统安装了 mcap 存储插件）：
>
>   ```bash
>   ros2 bag record --storage mcap -o bags/week0_demo_mcap /turtle1/pose /turtle1/cmd_vel /tf /tf_static
>   ```
>
>   如果报找不到 storage plugin，就 apt 装 `ros-humble-rosbag2-storage-mcap` 再试。

````md
A minimal “presentable” ROS2 system built on top of turtlesim:
- Topic: publish `/turtle1/cmd_vel`, subscribe `/turtle1/pose`
- Service: call turtlesim `/reset`, and custom `/week0/get_pose`

- rosbag2: record & replay key topics

```mermaid
flowchart LR
  subgraph UI[turtlesim window]
    T[turtlesim_node]
  end

  PL[pose_logger] -->|sub /turtle1/pose| T
  GPS[get_pose_service] -->|sub /turtle1/pose| T

  AS[drive_square_server] -->|pub /turtle1/cmd_vel| T
  CLI2[ros2 action send_goal] -->|action /week0/drive_square| AS

  TF[tf_broadcaster] -->|pub /tf| TF2[(tf tree)]
  TF -->|sub /turtle1/pose| T
````
